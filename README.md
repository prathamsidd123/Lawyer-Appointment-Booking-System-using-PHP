# Law-Appointment-Booking-System-Php
⚖️ Law Appointment Booking System

A Web-based Law Appointment Booking System developed using PHP & MySQL that allows users to book appointments with lawyers online, manage cases, payments, and generate appointment reports.

📌 Project Overview

The Law Appointment Booking System simplifies communication between clients and lawyers by providing an online platform for:

User Registration & Login
Lawyer Profile Management
Appointment Booking
Payment Management
Case Review System
PDF Report Generation

🚀 Features

👤 User Module
Register & Login
View Lawyers
Book Appointment
Make Payment
Give Review & Rating
Download Case Report (PDF)

⚖️ Lawyer Module
Manage Profile
View Appointments
Accept / Reject Cases
View Client Details

🛠 Admin Module
Manage Users
Manage Lawyers
Monitor Appointments
View Payments
System Control

🧩 Database Modules MYSQL
Users
Lawyer Profiles
Appointments
Payments
Reviews

🔐 User Roles
| Role   | Access              |
| ------ | ------------------- |
| Admin  | Full System Control |
| Lawyer | Case Management     |
| User   | Appointment Booking |


💻 Technologies Used
| Technology | Usage                 |
| ---------- | --------------------- |
| HTML       | Structure             |
| CSS        | Styling               |
| JavaScript | Frontend Interaction  |
| PHP        | Backend Logic         |
| MySQL      | Database              |
| FPDF       | PDF Report Generation |
| XAMPP      | Local Server          |

📄 PDF Report Feature

The system generates downloadable Case Reports using:
Appointment Status
Dynamic Appointment Data
Reviews & Ratings
